Description
The IxaGrupPortal is a web application built with React that helps manage an Airbnb business. The application allows hosts to manage their properties, reservations, and guests all in one place. It also provides a user-friendly interface for guests to view properties and make reservations.

Features
Property management: hosts can add, edit, and delete properties, as well as set pricing and availability.
Reservation management: hosts can view and manage their upcoming reservations, as well as approve or decline guest requests.
Guest management: hosts can view their guest history, including past reservations and reviews.
Property search: guests can search for available properties based on location, price, and other criteria.
Reservation booking: guests can make reservations for available properties directly through the application.
Payment processing: the application integrates with a payment gateway to process guest payments securely.

Installation
To install the IxaGrupPortal application, follow these steps:

Clone the repository to your local machine.
Install the necessary dependencies by running npm install.
Start the development server by running npm start.

Technologies
The IxaGrupPortal application is built with the following technologies:

React: a popular front-end JavaScript library for building user interfaces.
Redux: a predictable state container for JavaScript apps.
Node.js: a back-end JavaScript runtime.
Express: a popular web framework for Node.js.
MySQL: a relational database management system.
Stripe: a payment gateway for processing online payments.
